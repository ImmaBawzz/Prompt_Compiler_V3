# Product Requirements Document

## Product
Prompt Compiler

## Category
Creator tooling / prompt infrastructure / translation engine / workflow seed product

## Core problem
Creators repeatedly lose quality and identity while translating one idea across different generative tools and packaging surfaces.

## Core solution
Accept one structured creative brief plus one brand/style profile and compile them into multiple target-specific outputs.

## Primary user
A multidisciplinary creator who:
- works across multiple creative AI tools
- cares about style consistency
- wants reusable systems instead of one-off prompts
- wants exportability and future automation

## MVP
- create or load a brief
- create or load a style profile
- compile multiple outputs
- preview diagnostics and scores
- export the artifact bundle locally
- run the same compilation from a CLI

## Non-goals for MVP
- provider execution
- live collaboration
- auth-heavy flows
- billing integration
- autonomous internet agents

## Product wedge
The wedge is not “AI assistance.”
The wedge is **creative translation with structure and style control**.

## Product expansion path
- profile libraries
- template packs
- scoring and refinement loops
- workspace sync
- workflow recipes
- release packaging
- publishing automation
- team workspaces
- credits and subscriptions
