# V3 Overview

V3 upgrades the starter into a more serious operating scaffold.

## Three major changes

### 1. Better execution model
The agent now has explicit boot sequence, task loop, quality gates, control commands, and a stronger definition of done.

### 2. Better architecture
The compiler engine, CLI, API, and extension are framed as one reusable system instead of disconnected pieces.

### 3. Better product shape
The repo now reflects a product that can plausibly grow from local tooling into a monetized creative workflow platform.
