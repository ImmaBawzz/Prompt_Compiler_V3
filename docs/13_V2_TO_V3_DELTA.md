# V2 → V3 Delta

## What changed materially

### Agent system
V3 adds a clearer boot sequence, quality gate, control commands, and a more actionable task board.

### Product shape
V3 treats the compiler as a reusable system with three delivery surfaces: extension, CLI, and API.

### Engineering discipline
V3 adds CI scaffolding, a CLI package, ADRs, richer schemas, and better export conventions.

### Strategic value
V3 is closer to a real product seed than a planning bundle.
