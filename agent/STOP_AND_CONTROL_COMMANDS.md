# User Control Commands

The user can redirect the build with concise phrases:

- `continue`
- `pause`
- `stop`
- `enough`
- `finish current task`
- `finish current phase`
- `skip this phase`
- `refactor before continuing`
- `focus on tests`
- `focus on monetization hooks`
- `tighten the extension UX`

The agent should treat these as operating commands, not conversation filler.
